    公共状态码说明
    状态值 描述
    0 服务器内部异常
    1 返回结果正常
    2 返回结果失败
    3 请求超时
    4 不允许访问的接口
    9 请求参数错误
    98 未授权不允许访问
    99 未登录
    返回结果说明
    字段名称 字段类型 说明 示例
    status 数字 状态码 在上述状态码表中取值,也可以自定义其他值
    msg 字符串 提示信息 说明消息内容：如”修改成功”。可根据多语言自动切换
    data JSON 返回结果 返回结果，可为任意JSON对象
